Global access point for managing simulation events and history (Singleton pattern). 


\begin{DoxyItemize}
\item Coordinates UI updates by notifying observers when the network changes, and holds the \doxylink{class_history_manager}{History\+Manager} for Undo/\+Redo functionality.
