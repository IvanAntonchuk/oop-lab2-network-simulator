\doxysubsection{Detailed Description}
Base decorator class for dynamically adding behavior to nodes (Decorator pattern). 


\begin{DoxyItemize}
\item Wraps a \doxylink{class_network_node}{Network\+Node} and delegates traffic processing to it, allowing subclasses to add pre-\/ or post-\/processing logic. 
\end{DoxyItemize}
